# Holded — Facturación (Ventas y Compras)

## Tipos de documentos de venta

| Tipo | Descripción | Contabiliza IVA |
|------|-------------|-----------------|
| **Factura** | Documento fiscal definitivo | Sí |
| **Factura rectificativa** | Corrección/abono de factura emitida | Sí (inverso) |
| **Presupuesto** | Oferta al cliente, no contabiliza | No |
| **Proforma** | Pre-factura sin efecto contable | No |
| **Albarán** | Documento de entrega, sin IVA en libros | No |
| **Factura recurrente** | Se genera automáticamente según periodicidad | Sí |
