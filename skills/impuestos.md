# Holded — Impuestos y Modelos Tributarios

## Modelo 303 — IVA trimestral

### Generar el modelo 303 en Holded
1. Contabilidad → Impuestos → Modelo 303
2. Seleccionar **período**: 1T, 2T, 3T, 4T (o mensual si está en SII)
3. Holded calcula automáticamente:
   - **IVA repercutido** (ventas): base imponible + cuota
   - **IVA soportado deducible** (compras): base + cuota
   - **Resultado**: diferencia a ingresar o a compensar
4. Revisar líneas — verificar que todas las facturas del período estén contabilizadas
5. **Exportar** en formato AEAT (.xml o .pdf para presentar)

### Causas comunes de discrepancias en el 303
- Facturas de venta sin tipo de IVA asignado → no aparecen en el modelo
- Facturas de compra marcadas como "no deducible" por error
- Operaciones intracomunitarias sin "inversión del sujeto pasivo"
- Facturas del período anterior contabilizadas en el trimestre actual
