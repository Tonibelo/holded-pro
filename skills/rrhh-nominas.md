# Holded — RRHH y Nóminas

## Configuración inicial RRHH

### Datos de empresa para SS
1. RRHH → Configuración → Empresa
2. Introducir: CIF, código de cuenta de cotización (CCC), delegación provincial de la TGSS
3. Añadir convenio colectivo si aplica (afecta a tablas salariales)

### Configuración de cuentas contables para nóminas
- RRHH → Configuración → Cuentas contables
- Asignar cuentas para: sueldos (640), SS empresa (642), SS trabajadores (465), retenciones IRPF (475)
- Sin esta configuración, las nóminas no contabilizan correctamente
