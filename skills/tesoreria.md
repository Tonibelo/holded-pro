# Holded — Tesorería

## Cuentas bancarias

### Añadir cuenta bancaria
1. Tesorería → Cuentas → + Nueva cuenta
2. Opciones:
   - **Conexión bancaria directa** (Open Banking): introduce IBAN → autenticar con el banco → sincronización automática de movimientos
   - **Manual**: añadir solo el nombre y IBAN, importar extractos manualmente
3. Asignar cuenta contable (ej. 572 para cuentas corrientes, 570 para caja)

### Bancos compatibles con sincronización automática
Holded conecta con la mayoría de bancos españoles vía Open Banking (PSD2): Santander, BBVA, CaixaBank, Sabadell, Bankinter, ING, N26, Revolut, entre otros.

## Conciliación bancaria

1. Tesorería → Conciliación → seleccionar cuenta y período
2. Holded muestra movimientos bancarios vs movimientos contables
3. Emparejar automáticamente o manualmente
4. Los no emparejados quedan pendientes

## Previsión de tesorería

- Tesorería → Previsión: cobros y pagos pendientes proyectados
- Útil para detectar tensiones de liquidez antes de que ocurran
